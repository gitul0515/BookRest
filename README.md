## 기타 (배운점)

바닐라 JavaScript만으로 SPA를 구현하면서,  
프론트엔드 프레임워크의 동작 원리와 개발 환경을 학습했습니다.  
  
  - Historary API를 활용한 동적 라우팅  
  - 상태 기반 렌더링  
  - 컴포넌트 단위 개발 및 재사용  
  - Webpack, Babel, Prettier 등 환경 세팅 및 번들링 후 배포
